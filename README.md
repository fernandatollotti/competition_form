# Formulário de Competição

## Descrição
Projeto formulário de inscrição para a competição de natação. Objetivo é informar em qual categoria o nadador irá competir.

## Ferramentas
<ul>
  <li>HTML 5</li>
  <li>CSS 3</li>
  <li>PHP</li>
</ul>

	* [X] Verifica se campo da idade é um número
